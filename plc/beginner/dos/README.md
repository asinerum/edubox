Simple DOS Approaches for Beginner

How to Run a DOS App?

1. You may need a DOS emulation app like [DosBox](https://www.dosbox.com/)

2. Launch DosBox, then mount your working directory to a logical drive

3. Run your DOS app in the working directory

How to Compile ASM Source Code to DOS App?

1. You may need an ASM compiler like [NASM](https://www.nasm.us/)

2. Compile your ASM Source Code as:

```ts
nasm -f bin -o hello.com hello.asm
```
